# Remidi Works Platform Demos

Decision Support Platform for Revenue Model Excellence.

**Target Audience:** Middle Market PE Firms & Portfolio Companies

## URLs

- `/` - Landing page with both demo options
- `/user` - Portfolio Company view (case study walkthrough)
- `/investor` - Investor Portfolio dashboard

## Core Value Proposition

- **WHAT:** See where you rank vs. peers (benchmark position)
- **SO WHAT:** Know what's broken (diagnosed gaps)
- **NOW WHAT:** Fix what matters (prioritized action)

## Company Demo Flow (7 Steps)

| Step | Feature |
|------|---------|
| 0 | Dashboard Home - Initial scores, preliminary gaps, observable data sources |
| 1 | AI Interview - Typing effect, 3 questions that dynamically adjust scores |
| 2 | Updated Assessment - Shows score changes, priority workstreams |
| 3 | Action Plan - 6-step guided workstream with locked/unlocked states |
| 4 | Executing Step 1 - Gap analysis vs best practice, competitor comparison |
| 5 | Pause & Resume - Saved progress demo |
| 6 | Why Remidi Works - Value props and close |
